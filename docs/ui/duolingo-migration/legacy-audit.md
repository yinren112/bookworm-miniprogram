# 旧样式残留审计（Duolingo 风格迁移）

说明：统计为“命中次数 / 涉及文件数”。优先处理可达页面（见 pages.md）。遗留未注册页面默认不纳入主链路验收，但也会影响“全站一致性”指标，后续可选择清理。

## 审计项与统计

| 模式 | 统计 | 备注 |
|---|---:|---|
| box-shadow | 0 / 0 | 迁移目标：小程序目录清零（禁止模糊阴影） |
| 旧色值 #2c5f2d | 0 / 0 | 迁移目标：清零（只允许 tokens/兼容别名出现） |
| .btn-primary | 0 / 0 | 无命中 |
| .clay- | 0 / 0 | 迁移目标：清零（避免两套风格混用） |
| 渐变（linear/radial/conic） | 0 / 0 | 迁移目标：清零或仅保留极少数非交互装饰（默认清零） |
| text-shadow / drop-shadow | 0 / 0 | 迁移目标：清零 |
